== サポートされていないアクション

以下のアクションは、NetApp Backup and RecoveryのHyper-Vワークロードではサポートされていません：

* 複数の Hyper-V ホストの VM を使用してリソース グループを作成する
* スパニングディスク（複数の CIFS 共有にまたがる）
* Hyper-Vの「プロセッサ互換性」設定に関わらず、異なるCPUベンダー（IntelからAMD、またはその逆）のシステム間でVMまたはVMデータを復元することはできません。この設定は、同一ベンダーの異なる世代間（例：IntelからIntel、AMDからAMD）の互換性のみをサポートします。
